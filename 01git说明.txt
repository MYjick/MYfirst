git 支持大量使用分支

分支相关的命令
1. 创建分支
git branch  分支名

2. 查看分支
git branch  (列出所有分支)

3. 切换分支
git checkout 分支名

4. 合并分支
git merge 分支名（shuihu）
合并分支到当前分支

5. git log --graph 以图片化方式显示提交记录
可以很清楚查看各分支合并的过程

如图形过长话：
 按enter 继续打印图形
 q 退出

6. 当合并发生冲突，手动处理之后 提交做成版本就是合并后版本

7. vim相关
命令行编辑器
h   光标向前
l   光标向后
j   光标向下
k   光标向上

i   进入插入模式（输入模式）

:wq  保存并退出
:q!  不保存退出
o     在当前行下插入新行并进入插入模式

vimtutor   vim 入门教程
v    可视化选区
d    删除可视化选区并记录删除的内容
p    粘贴内容
u    撤消
